Directory: src/
Description: Contains the following OSNAP LOST web app demo source files.

1.  index.html - OSNAP LOST user login page, used as home page
2.     rf.html - "Report Filter" page, lets user request desired report type
3.    fir.html - "Facility Inventory Report" page, lists all assets at specified facility
4.    itr.html - "In-Transit Report" page, lists all assets on specified route
5. logout.html - User logout page, prompts to return to home
